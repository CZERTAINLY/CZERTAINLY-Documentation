# Sample Properties

The following are sample properties that configures worker for augmentation of XAdES signature to B-LTA:

```
WORKERGENID1.TYPE=PROCESSABLE
WORKERGENID1.IMPLEMENTATION_CLASS=com.czertainly.signserver.module.xades.augmentor.XAdESAugmentor
WORKERGENID1.NAME=XAdES-Baseline-LTA-Augmentor
WORKERGENID1.SIGNATURE_LEVEL=XAdES-BASELINE-LTA
WORKERGENID1.AUTHTYPE=org.signserver.server.UsernameAuthorizer
WORKERGENID1.ACCEPT_ALL_USERNAMES=true
WORKERGENID1.EMBED_CRL=true
WORKERGENID1.EMBED_OCSP=true
WORKERGENID1.DOWNLOAD_VALIDATION_DATA_FOR_UNTRUSTED_CHAINS=true
WORKERGENID1.TSA_URL=https\://freetsa.org/tsr
WORKERGENID1.TSA_USERNAME=test
WORKERGENID1.TSA_PASSWORD=test
WORKERGENID1.TSA_DIGESTALGORITHM=SHA512
WORKERGENID1.PROCESSINTRANSACTION=true
```
